更新内容

  • 新的稳定滤波器内核。
  • 内置音序器扩展为左右每侧 64 步，超过 64 步时从最早步替换。
  • 新增啸叫抑制功能，会在回授出现时降低增益与延音，以减少过载风险。

操作说明

  1. 使用桌面版 Chrome 或 Edge，通过 HTTPS 打开此页面。请确保使用能正常传输数据的 USB 线。银色 Wingie2 需要使用 USB A–C 线。
  2. 连接 Wingie2，并关闭串口监视器、配置页等占用串口的软件。
  3. 点击“连接 Wingie2 / Connect”，在弹出的列表中选择 Wingie2 的 USB 串口。
  4. 显示“设备已连接”后，点击“安装固件 / Install”。安装期间不要断开 USB 或关闭页面。
  5. 显示“安装完成”后即可使用。MIDI、调律、Cave 与 Ratio 设置会保留。

Changelog

  • A new stable resonator filter core.
  • The built-in sequencer now supports 64 steps per side, replacing the oldest step first when more steps are entered.
  • New feedback suppression lowers gain and sustain when feedback builds, reducing overload risk.

Instructions

  1. Open this page over HTTPS in desktop Chrome or Edge. Make sure you use a USB cable that supports data transfer. Silver Wingie2 units require a USB-A-to-USB-C cable.
  2. Connect Wingie2, and close serial monitors, configuration pages, or other software using the port.
  3. Select “连接 Wingie2 / Connect”, then choose the Wingie2 USB serial port from the list.
  4. When the page says the device is connected, select “安装固件 / Install”. Do not unplug USB or close the page during installation.
  5. Wingie2 is ready when the page reports completion. MIDI, tuning, Cave, and Ratio settings are retained.
